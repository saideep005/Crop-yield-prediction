# 🌾 Crop Yield Prediction using Machine Learning

## 📌 Project Overview
This project predicts rice crop yield based on environmental factors such as rainfall, temperature, humidity, fertilizer usage, and soil type. It uses an XGBoost model and provides results through a Flask web application.

---

## ⚙️ Features
- Predict crop yield (tons/hectare)
- Input: rainfall, temperature, humidity, fertilizer, soil type
- Graph visualization using Chart.js
- Login system
- Simple UI

---

## 🛠️ Technologies Used
- Python
- Flask
- XGBoost
- HTML, CSS
- JavaScript (Chart.js)

## ▶️ How to Run
1. Install dependencies:
pip install flask numpy pandas scikit-learn xgboost

2. Run:
python app.py

3. Open:
http://127.0.0.1:5000/
